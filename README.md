A dynamic movie search application built with **React**, integrating the **OMDb API** to retrieve real-time movie data. The app displays movies in a horizontal scrolling layout and allows users to add or remove favourites, which are saved using **localStorage** for persistent storage.

## üöÄ Features

- **Movie Search** ‚Äì Fetch movies from the OMDb API based on user input.
- **Add to Favourites** ‚Äì Save selected movies to a favourites list.
- **View Search & Favourites** ‚Äì Display search results and favourite movies in a responsive UI.
- **Remove from Favourites** ‚Äì Delete movies from the favourites list.
